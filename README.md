## What is this tool?

Codex-loop manages your automated tasks on Windows. It ensures your specific processes stay active until they hits a set goal. You can define targets based on time or total rounds of execution. 

Many users find that manual task management takes too much time. This software automates that repetition. It monitors your Codex processes and restarts them if they stop before your target criteria. It provides a simple way to keep your workflow moving without manual intervention.

## ⚙️ Using the software

Open the application using the desktop icon. The main window shows a list of your current tasks. 

**Adding a task**
Click the Add Task button at the top of the screen. A new window asks for the path to your Codex task. Use the Browse button to locate the file on your computer.

**Setting completion targets**
You must tell the program when to stop. Select one of the two modes:
- Time-based: Set the number of hours and minutes you want the task to run. The loop ends when the timer reaches zero.
- Round-based: Enter the number of cycles you want the process to repeat. The loop ends when the counter hits your number.

**Starting the execution**
Click the Start button after you configure your task. The software creates a loop. You can monitor the progress on the dashboard. The application shows the time remaining or the current round count.

## 🧩 Features

- Persistent loop monitoring: The program watches your task. If the Codex process crashes, the app restarts it.
- Resource management: The tool keeps your system stable during long operations. 
- Auto-shutdown: The system closes the task automatically once you reach your target.
- Task logs: The app generates a report at the end of each run. You can find this file in the installation folder.
- Lightweight design: The software uses low system memory. It runs in the background while you focus on other work.

## 🔧 Solving common issues

**The application does not open**
Check if you have an older version running in the background. Right-click the icon in your system tray and select Exit. Try starting the program again.

**The task stops early**
Check your input settings. Ensure the target path is correct. If the task requires a login or specific permissions, the software might fail to restart the process.

**The window freezes**
Click the Refresh button on the dashboard. If the issue continues, restart the application. Your progress saves to the local log file even if the interface stops responding.

**Installation errors**
Ensure you have enough space on your hard drive. If the installer asks for administrator permission, provide it to allow the software to create the necessary shortcut files.
